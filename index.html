About me

I am To Thanh Tung, a high-school student interested in the intersection of technology, finance, embedded systems, and student leadership. My portfolio is built around the idea that a strong academic profile should show not only achievements, but also how I think, build, test, present, and improve a project from beginning to end.

Through the Duino-Coin project, I explored how a small ESP32 device can connect to a real network service, process mining tasks, and send data to a monitoring dashboard. The project helped me practice system thinking: hardware setup, Wi-Fi communication, API data, dashboard visualization, report writing, and presentation preparation.

Outside technical work, I actively participate in school clubs, charity activities, martial arts events, and student council programs. These experiences help me build communication skills, teamwork, responsibility, and the confidence to contribute in both academic and community environments.
